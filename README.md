
GAMES

1) LASER : operation check 動作確認用
2) POV   : AVT15 の表示
3) GAME(1) High & ROW : put the LED lights(1~8). You think next value is High/LOW than display value. 1〜8の値をLED点灯。次の値が高いか低いかを予想。
4) GAME(2) Not Binary : 2進数を1~255まで表示。間違ったBinaryの際に　push the action swtich.
5) GAME(3) Symon Says : 2進数を複数パターン表示。表示された値を10進数に直して、その分だけpush the action switch.
